Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag </strong>. Nunca use a tag </b> para colocar destaque em alguma palavra ou frase pois não é uma tag semântica.

Itálico / Ênfase

Aqui temos um termo em itálico usando a tag </em>. Nunca use use a tag </i> para colocar ênfase em uma palavra ou frase pois não é uma tag semântica.

Podemos também criar um texto marcado usando a tag </mark>.

Aqui temos mais um texto marcado.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Nunca use a tag </big> para deixar uma letra ou frase grande pois não é uma tag semântica. Já a tag </small> pode ser usada pois é uma tag com semântica (isso porquê letras pequenas são usadas em contratos, termos, etc. Assim dando sentido ao uso da tag).

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Nunca use a tag </u> para sublinhar uma frase pois essa não é semântica.

Texto sobrescrito

Para inserir letras ou números elevados como essa equação aw2+24ay3-45aw2, use a tag </sup>

Texto subscrito

Para deixar números ou letras rebaixados como H2O, use a tag </sub>
